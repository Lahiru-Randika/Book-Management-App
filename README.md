# Book Management App

A simple full-stack web application to manage books using Angular and ASP.NET Core Web API.

## Features
- Add books
- View books
- Edit books
- Delete books

## Tech Stack
- Frontend: Angular
- Backend: ASP.NET Core Web API
- Storage: In-memory list

## How to Run

### Backend
```code
cd BackendAPI
dotnet run
```

### Frontend
```code
cd frontend
ng serve
```
